Welcome to HJCFIT’s documentation!¶

The goal of HJCFIT is to provide a collection of tools for scientific research on ion channels. The package is derived from the DCPROGS suite and consists of a C++ implementation of the Likelihood calculations along with Python wrappers. The code is a port of Fortran code with ~30 years usage at University College London. The rationale is to preserve and cultivate these tools for future research applications.

For a description of the methods involved, see [CH82], [HJC92], [CH95b], [CH95a], [CHS96].
